下载 App Store 上的 IPA 文件到 Mac

发表于 2020-05-23 阅读次数： Disqus：

前言

苹果已经从iTunes的12.7版本中删除了App Store，这使得从App Store下载IPA变得更加困难。

但是，现在可以通过使用Apple Configurator 2来实现。

Apple Configurator 2

选中设备，点击 Add -> Apps

如果设备上之前没有当前添加的 App，会直接安装。

如果该 App 已经再设备上存在，会询问是否覆盖，此时不要做选择。

打开 使用 iTerm 或者 PopClip打开进去如下路径：

应该看到带有随机生成的ID的目录，深入到该目录应该就能找到IPA文件了。
